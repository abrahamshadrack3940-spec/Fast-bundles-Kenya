# Fast Bundles Kenya Website

A simple bundles selling website using HTML, CSS, JavaScript and Node.js.

## Features
- Bundle pricing cards
- M-PESA Till Number payment instructions
- WhatsApp contact button
- Responsive design
- Simple Node.js server

## Files
- index.html
- styles.css
- script.js
- server.js

## How To Run

1. Install Node.js
2. Open terminal in project folder
3. Run:

npm install express

4. Start server:

node server.js

5. Open browser:

http://localhost:3000
